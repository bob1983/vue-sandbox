# Unit test vue

## What to test

### Component
#### Should be testing 
- Inputs
    - data()
    - Props
    - User interaction
    - Lifecycle methods
    - Store
    - Route params
- Outputs
    - Rendered dom
    - function calls
    - emitted events
    - Store
    - Changes in child components

#### Should not be testing
- Implementation details
- Framework itself
    - prop validation
    - ...
