# Vue 3 ( with composition-api & TypeScript ) getting started

Vue 3 を TypeScript + composition-api 使用で開発するためのサンプル

## Project setup

```shell
npm install
```

### Compiles and hot-reloads for development

```shell
npm run serve
```

### Run your unit tests

```shell
npm run test:unit
```

### Lints and fixes files

```shell
npm run lint
```
